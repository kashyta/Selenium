
DemoMavenEclipseProject - Created an automation flow to register an email id and create an 
account for login into a web application using BDD framework creating a maven prject which 
can be further integrated with Jenkins

System Requirements:
  MAVEN install and configure

  Create a maven project
  Specify the Group Id & Artifact Id 
  Define dependencies in POM.xml
  Create test cases in test under src
  Run test as maven test
